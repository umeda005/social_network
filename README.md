# social_network
API-приложение мини-социальной сети, разработанное с использованием Python и FastAPI.

# Описание проекта
Данный проект представляет собой серверное API-приложение, предназначенное для реализации базового функционала социальной сети.
API обеспечивает взаимодействие пользователей, публикацию постов и работу с лайками, а также поддерживает механизм аутентификации и авторизации.
Проект ориентирован на демонстрацию принципов построения REST-API, обработки HTTP-запросов и организации серверной логики веб-приложения.

# Используемые технологии
Python , FastAPI, SQLAlchemy, Pydantic, Uvicorn, JWT (JSON Web Token)

# Основные возможности 
Регистрация и аутентификация пользователей

Авторизация с использованием JWT-токенов

Создание и просмотр постов

Получение списка постов

Добавление лайков к постам

Работа с пользователями

Автоматическая генерация документации Swagger (OpenAPI)
